Pour créer une sauvegarde, utilisez le menu "Fichier / Créer une sauvegarde" puis sélectionnez le répertoire et le nom du fichier que vous souhaitez utiliser.

Ce fichier de sauvegarde peut être ensuite importé à nouveau dans BudgetView à partir du menu "Fichier / Restaurer une sauvegarde".

NB:Une sauvegarde effectuée sur une plateforme (Mac/Windows/Linux) peut être restaurée sur toutes les autres plateformes.
