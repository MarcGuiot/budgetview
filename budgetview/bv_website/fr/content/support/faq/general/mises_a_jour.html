A chaque lancement, BudgetView vérifie automatiquement la présence d'une nouvelle version. S'il en trouve une, vous verrez apparaître sur la page d'accueil de l'application, environ 30 secondes après le lancement, un message vous proposant de relancer BudgetView.

Pour que ce mécanisme de mise à jour automatique fonctionne, il faut toutefois que BudgetView puisse accéder à Internet. Si la mise à jour ne se fait pas, vous pouvez vérifier qu'aucun pare-feu ne limite l'accès Internet pour cette application.

Pour connaître la version de l'application installée sur votre ordinateur, vous devez vous rendre dans le menu "Aide / A propos".

L'historique des versions de BudgetView est décrit ici :

https://www.mybudgetview.fr/support/derniers-changements

Si vous le souhaitez, vous pouvez forcer la mise à jour en réinstallant BudgetView manuellement :

https://www.mybudgetview.fr/telechargement
