Pour suspendre votre abonnement vous devez supprimer votre compte BudgetView Sync

A l'issue de cette suppression :

  • Toutes les données vous concernant sur nos serveurs sont supprimées
  • Les paiements de votre abonnement sont interrompus
  • Toutes les données déjà présentes sur votre ordinateur restent bien entendu inchangées.

Vous pouvez à tout moment créer un compte BudgetView Sync à nouveau.

Note: Si vous n'avez plus accès à BudgetView pour effectuer la suppression de votre compte, nous vous invitons à nous écrire à l'adresse support@mybudgetview.fr en indiquant l'adresse email que vous avez utilisée pour créer votre compte Sync.
