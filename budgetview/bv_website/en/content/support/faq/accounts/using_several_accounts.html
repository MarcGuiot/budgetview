You can use several accounts in BudgetView.

This is true for both types of accounts:

  1. You can manage as many bank accounts as you want for a given budget. You just have to import the corresponding transactions, or create these accounts in the Accounts view.
  2. You can create several user accounts in BudgetView, to manage different budgets or allow for different users on the same computer. To create a new user account, use the "File / Go to login screen" menu.
