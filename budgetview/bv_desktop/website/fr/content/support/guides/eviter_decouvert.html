Pour éviter de passer dans le rouge, il vous faut consulter BudgetView régulièrement et vérifier les soldes des comptes qui se trouvent sur la barre de gauche de l'application :

Dans cet exemple :

  • Le solde minimum est de 301€ au mois de mai
  • Le solde minimum est de -1420€ au mois de juin

Vous devez donc intervenir à l'avance pour éviter que ce compte ne passe dans le rouge le mois prochain. Vous pouvez par exemple :

  • Réduire le montant prévu d'une de vos enveloppes (typiquement les loisirs et les courses, mais cela peut aussi être une dépense importante que vous remettez à plus tard)
  • Effectuer un virement à partir d'un autre de vos comptes

NB: Pour que ces informations soient aussi fiables que possible, vous devez vous assurer d'avoir bien catégorisé vos opérations et d'avoir convenablement réglé vos enveloppes.
