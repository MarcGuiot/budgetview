La vue Budget vous donne deux informations complémentaires pour savoir combien vous pouvez dépenser dans une enveloppe :

Montant restant sur l'enveloppe

Le premier élément à prendre en compte est le montant que vous avez prévu pour l'enveloppe en question :

Dans cet exemple il vous reste environ 800 - 120 = 680€ à dépenser en Courses pour ce mois, tandis que vous avez déjà dépassé le montant que vous vous etiez fixé pour l'argent liquide.

Note: Pour faciliter la lecture de cet écran, nous prévoyons d'afficher le montant restant plutôt que le montant réel dans une prochaine version de l'application.

Montant restant sur le compte

Le second élément est le solde minimum de votre compte :

Dans cet exemple vous pouvez encore dépenser 301€ en plus avant de passer dans le rouge. Notez toutefois que le mois suivant sera davantage dans le rouge encore ! C'est là que les outils atteignent leurs limites : vous seul pouvez prendre la décision sur la base des informations qui vous sont fournies. 8^)
