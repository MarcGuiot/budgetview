Pour déplacer le répertoire de données de BudgetView :

  • Sélectionnez l'option "Fichier / Préférences..."
  • Rendez-vous dans l'onglet "Stockage" et suivez les instructions affichées.
    		•

    Note : si vous utilisez un système de partage de fichiers tels que Dropbox ou Google Drive, assurez-vous qu'un seul BudgetView n'y accède à la fois pour éviter toute perte de données.
