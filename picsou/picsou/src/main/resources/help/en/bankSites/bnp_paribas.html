Suivez les instructions ci-dessous pour télécharger vos relevés sur le site de votre banque :

Accédez à votre espace client

Rendez-vous sur le site internet de BNP Paribas :

http://www.bnpparibas.net

Sur la page d'accueil du site, cliquez sur le lien "Accès aux comptes" 

Identifiez-vous et accédez à la liste de vos comptes

Accédez à la page Services

Dans les menus de gauche, vous trouverez un entrée Services :

Accédez à la page de téléchargement

Dans le bloc "Comptes / Services", vous trouverez un lien vers la page de téléchargements :

Lancez le téléchargement

Choisissez les paramètres suivants :

Format

QIF Configuration personnelle

Vous pouvez laisser les formats de dates et de nombres sur JJ/MM/AA et (.)
Période Toutes vos opérations
Mise en mémoire Vous pouvez sélectionner "Oui" pour ne pas avoir à choisir à nouveau ces paramètres dans le futur.

Utilisez la fonction "Ouvrir avec" de votre navigateur

Au moment de lancer le téléchargement, votre navigateur va vous proposer soit de sauver le fichier sur votre disque dur, soit de l'ouvrir directement avec une application. Choisissez "Ouvrir avec" en sélectionnant BudgetView - le fichier sera alors directement sélectionné dans l'application.
