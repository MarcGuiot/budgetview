Dans le cas où vous ne pouvez pas importer vos opérations à partir du site internet de votre banque, il vous est possible de saisir vous-mêmes vos opérations une à une dans l'outil. Cette approche est également utile pour gérer un compte "argent liquide", pour lequel - par définition - l'import est impossible.

Pour cela, vous devez vous rendre dans la vue Catégorisation (sélectionner). Au bas du bloc de gauche, vous trouverez un lien "Saisir des opérations manuellement".

Pour saisir des opérations, vous devez disposer d'un compte dont le mode de mise à jour soit "Saisie manuelle". Si ce n'est pas le cas, il vous sera proposé d'en créer un.
