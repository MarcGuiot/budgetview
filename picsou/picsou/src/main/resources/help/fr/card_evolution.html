La vue Evolution offre une vue d'ensemble sur les montants prévus pour les enveloppes de chaque poste budgétaire, sur une période de plusieurs mois.

Grâce à cette vue, vous pouvez :

  • explorer la composition de votre budget.
  • voir l'évolution de ce budget dans les mois à venir, et ainsi anticiper d'éventuels problèmes ou préparer des projets.
  • comparer le budget du mois sélectionné à celui du mois précédent.

Vous pouvez cliquer sur le montant d'une enveloppe pour éditer les caractéristiques de l'enveloppe - typiquement pour modifier le montant attendu, ou pour changer la périodicité de l'enveloppe.
