Pour obtenir des estimations réalistes dans les mois à venir, vous devez régler convenablement toutes les enveloppes qui constituent votre budget.

Suivez le guide ci-dessous pour améliorer les estimations :

Classez l'ensemble de vos opérations

Vérifiez que vous avez bien classé la grande majorité des opérations que vous avez importé jusque là.

Ajustez vos enveloppes

Assurez-vous que des enveloppes sont prévues pour toutes les dépenses à venir, et que vous avez bien précisé dans BudgetView la façon dont elles se répétent.

Voici une liste des enveloppes les plus fréquemment ajustées :

  • Les anniversaires de vos proches, qui se répètent... tous les ans.
  • Les révisions de vos véhicules, qui se répetent habituellement tous les ans.
  • Les factures de gaz et d'électricité, qui se répetent habituellement tous les deux mois.
  • Les impôts, dans le cas où vos règlements ne sont pas mensualisés
  • Certaines assurances, par exemple les assurances habitation, qui se répètent tous les ans
  • Les cadeaux de Noël
  • Les revenus irréguliers (13e mois, primes, etc.)

Ajustez votre enveloppe "Divers"

Vous pouvez anticiper un montant pour les imprévus, en utilisant votre enveloppe "Divers" et en réglant une somme approximative.

Importez vos comptes d'épargne

Assurez-vous d'avoir bien déclaré vos comptes d'épargne dans BudgetView. Si après tous les réglages précédents le solde minimum de vos comptes dans les mois à venir est largement positif, vous pouvez prévoir de renforcer votre épargne.
