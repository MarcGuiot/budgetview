Avant d'utiliser les informations de cette page, assurez-vous que vous avez bien pris connaissance des notions de série et de postes budgétaires.

Revenus exceptionnels

Si vous percevez des revenus à titre exceptionnel (dons de proches, primes, treizièmes mois...), nous vous recommandons de créer une série "Revenus exceptionnels" dans le poste budgétaires "Revenus". Vous pourrez passer cette opération en périodicité "Irréguliere" plutôt que "Tous les mois", pour éviter que les montants ne soient propagés aux mois suivants. Vous pourrez également la passer en mode "budget manuel" pour pouvoir saisir vous-mêmes les montants attendus dans les mois à venir.

Remboursements de santé

Pour gérer vos dépenses de santé et les remboursements associés, nous vous recommandons de créer une série unique "Santé" dans le poste budgétaire "Enveloppes", et d'y affecter toutes ces opérations. Les dépenses et les remboursements pourront alors s'équilibrer.

Argent liquide

Si vous recevez fréquemment de l'argent en liquide, vous pouvez créer un compte "Liquide" dédié, et placer ce compte en mode "Saisie manuelle". Vous pourrez alors saisir vous-même vos opérations en liquide.

Pour les retraits d'argent liquide, vous pouvez répartir les montants pour faire apparaître des achats particuliers. Si vous ne souhaitez pas rentrer trop dans le détail de ces dépenses, vous pouvez créer une série de type "Enveloppe" pour y placer toutes ces opérations.
