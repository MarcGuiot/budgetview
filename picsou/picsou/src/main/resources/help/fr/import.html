BudgetView utilise les relevés bancaires électroniques que vous fournit votre banque.

Lorsque vous sélectionnez votre banque dans la fenêtre d'import :

  • Si votre banque le permet, BudgetView vous proposera de saisir directement vos identifiants pour télécharger automatiquement vos relevés.
  • Dans les autres cas, vous devez vous rendre sur le site internet de votre banque pour y télécharger vous mêmes vos relevés (en savoir plus).

Si votre banque ne propose aucun mode de téléchargement, vous pouvez saisir vos opérations manuellement.

Pour plus d'informations, veuillez consulter la page Importer vos relevés d'opérations sur notre site de support.
