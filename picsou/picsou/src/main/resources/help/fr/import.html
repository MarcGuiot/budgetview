CashPilot utilise les relevés bancaires électroniques que vous fournit votre banque.

Si votre banque ne propose pas ces téléchargements, vous pouvez saisir vos opérations manuellement.

Télécharger vos relevés

Pour télécharger ces relevés, vous devez disposer d'un accès au site internet de votre banque. La séquence d'import se déroule alors comme suit :

  • Vous vous connectez au site internet de votre banque
  • Vous vous rendez sur la page de téléchargement du site
  • Vous sélectionnez les comptes, la période et le format d'export souhaités (ces notions sont détaillées ci-dessous)
  • Vous lancez le téléchargement proprement dit
  • Si votre navigateur vous propose un choix entre "Ouvrir avec CashPilot" et "Enregistrer sous", vous pouvez choisir "Ouvrir avec CashPilot" : le fichier sera alors automatiquement redirigé vers CashPilot, où vous pourrez poursuivre l'import.
  • Si vous n'avez pas opté pour l'option "Ouvrir avec", vous devez enregistrer le fichier de relevés sur votre ordinateur, puis l'ouvrir à partir de CashPilot.

Banques et formats supportés

CashPilot fonctionne avec les principales banques Françaises, mais aussi avec la plupart des banques qui proposent un service de téléchargement en ligne.

CashPilot reconnaît les deux formats de fichiers les plus couramment utilisés :

  • OFX: Il s'agit du format proposé pour des logiciels tels que Microsoft Money. Nous vous recommandons d'utiliser ce format, qui contient les informations de compte et de solde.
  • QIF: Il s'agit du format géré par le logiciel Quicken d'Intuit. Il ne doit être utilisé qu'en absence de fichiers OFX, car il ne contient pas d'informations sur le compte des opérations téléchargées, en particulier les soldes de ces comptes.

Import initial

Dans un premier temps, importez les opérations de vos comptes chèques uniquement. Vous pourrez récupérer vos autres comptes par la suite.

Pour établir un budget réaliste, importez environ un ou deux mois complets de données . Il n'est pas utile d'importer une plage plus large, cela vous donnera plus de travail de catégorisation pour un bénéfice limité.
