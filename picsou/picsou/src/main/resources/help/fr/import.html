BudgetView utilise les relevés bancaires électroniques que vous fournit votre banque.

Note: Si votre banque ne propose pas ces téléchargements, vous pouvez saisir vos opérations manuellement.

Pour télécharger ces relevés, suivez la séquence suivante :

1. Connectez-vous au site de votre banque

Recherchez votre banque dans la fenêtre d'import. Si celle-ci propose un mode de téléchargement direct (sans passer par leur site internet), BudgetView vous proposera de saisir vos identifiants de connexion et de procéder directement au téléchargement. Il vous suffira alors de suivre les instructions à l'écran.

Si votre banque ne propose pas de téléchargement direct, vous devez vous rendre sur son site internet et suivre les instructions ci-dessous.

Note: Si votre banque ne figure pas dans la liste des banques proposées, il vous suffit d'utiliser votre navigateur internet pour vous rendre par vous-même sur le site de votre banque. BudgetView fonctionnera correctement avec votre banque, pour peu que celle-ci propose bien un téléchargement au format OFX ou QIF.

Une fois sur le site de votre banque, entrez vos identifiants de connexion, puis rendez-vous sur la page de téléchargement des relevés.

2. Préparez le téléchargement

Vous devrez indiquer les informations suivantes :

Format

BudgetView reconnaît les deux formats de fichiers les plus couramment utilisés :

  • OFX: Il s'agit du format proposé pour des logiciels tels que Microsoft Money. Nous vous recommandons d'utiliser ce format, qui contient les informations de compte et de solde.
  • QIF: Il s'agit du format géré par le logiciel Quicken d'Intuit. Il ne doit être utilisé qu'en absence de fichiers OFX, car il ne contient pas d'informations sur le compte des opérations téléchargées, en particulier les soldes de ces comptes.
Période des opérations

Vous pouvez télécharger les opérations des deux ou trois derniers mois sans vous soucier des données déjà importées dans l'outil - BudgetView saura éviter d'importer des opérations en doublon.

Pour établir un budget réaliste, importez environ un ou deux mois complets de données  lors de la première utilisation.
Format de dates

Dans la plupart des cas BudgetView saura reconnaître les formats de dates par lui-même. Il n'est donc pas utile de se soucier de cette information. En cas de problème spécifique, choisissez le format JJ/MM/AAAA.

Comptes

Dans un premier temps, importez les opérations de vos comptes chèques uniquement. Vous pourrez récupérer vos autres comptes par la suite.

3. Lancez le téléchargement

Si votre navigateur vous propose un choix entre "Ouvrir avec BudgetView" et "Enregistrer sous", vous pouvez choisir "Ouvrir avec BudgetView" : le fichier sera alors automatiquement redirigé vers BudgetView, où vous pourrez poursuivre l'import.

Pour plus d'informations, veuillez consulter la page Importer vos relevés d'opérations sur notre site de support.
