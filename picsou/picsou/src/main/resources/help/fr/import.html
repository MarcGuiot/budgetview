Fourmics s'appuie sur les relevés bancaires éléctroniques que vous fournit votre banque.

Pour télécharger ces relevés, vous devez disposer d'un accès au site internet de votre banque. La séquence d'import se déroule alors comme suit:

  • Vous vous connectez au site internet de votre banque
  • Vous vous rendez sur la page de téléchargement
  • Vous sélectionnez les comptes et la période souhaités, ainsi que le format d'export (ces notions sont détaillées ci-dessous)
  • Vous lancez le téléchargement proprement dit
  • Si votre navigateur vous propose un choix entre "Ouvrir avec Fourmics" et "Enregistrer sous", vous pouvez choisir "Ouvrir avec Fourmics" : le fichier sera alors automatiquement redirigé vers Fourmics, où vous pourrez poursuivre l'import.
  • Si vous n'avez pas opté pour l'option "Ouvrir avec", vous devez enregistrer le fichier de relevés sur votre ordinateur, puis l'ouvrir à partir de Fourmics.

Fourmics fonctionne avec les principales banques Françaises, mais aussi avec la plupart des banques qui proposent un service de téléchargement en ligne.

Fourmics reconnaît les deux formats de fichiers les plus couramment utilisés :

  • OFX: Il s'agit du format proposé pour des logiciels tels que Microsoft Money. Il est recommandé d'utiliser ce format, qui contient les informations de compte et de solde.
  • QIF: Il s'agit du format géré par le logiciel Quicken d'Intuit. Il ne doit être utilisé qu'en absence de fichiers OFX, car il ne contient pas d'informations sur le compte des opérations téléchargées, en particulier les soldes de ces comptes.

Dans un premier temps, vous pouvez vous contenter d'importer les opérations de vos comptes chèques. Vous pourrez récupérer vos autres comptes par la suite.

Pour établir un budget réaliste, il est recommandé d'avoir au minimum deux mois de données.
