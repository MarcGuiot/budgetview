BudgetView gère quatre types de comptes différents :

Comptes courants

Il s'agit typiquement des comptes chèque avec lesquels vous faites vos opérations courantes.

Vous pouvez catégoriser les opérations de ces comptes dans l'ensemble des postes budgétaires : revenus, fixe, variable, etc.

Comptes d'épargne

Il s'agit habituellement des livrets A, Codevi, PEL, PEA, etc. D'une manière plus générale, il s'agit de comptes sur lesquels vous ne faites pas d'opérations courantes.

Les opérations d'un compte d'épargne ne peuvent être catégorisées que dans le poste Epargne. Si vous utilisez l'un de ces comptes pour effectuer des opérations que vous souhaitez voir dans votre budget (courses, loisirs, loyer, etc.) vous devez déclarer le compte en tant que Compte Courant.

Comptes carte à débit différé

Dans le cas des cartes à débit différé :

  • les opérations carte sont prélevées sur un compte dédié
  • une fois par mois, le solde du compte carte est remis à zéro et une opération de prélèvement du montant total des opérations du compte carte apparaît sur le compte principal.

Consultez la page cartes à débit différé pour en savoir plus.

Cartes de crédit avec intérêts

Ces comptes représentent une réserve de crédit renouvelable, avec le remboursement des dépenses à la banque étalé par petites mensualités pour l’utilisateur. Ces comptes sont gérés comme des comptes courants dans BudgetView.
