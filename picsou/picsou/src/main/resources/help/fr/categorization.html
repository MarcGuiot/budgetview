Pour calibrer votre budget, vous devez créer les séries qui le caractérisent et y affecter les opérations que vous téléchargez du site internet de votre banque.

Dans la vue Catégorisation, Fourmics vous présente toutes les opérations que vous devez classer. Vous devez sélectionner ces opérations, puis les affecter à des séries parmi six postes budgétaires distincts :

  • Les revenus
  • Les frais fixes
  • Les enveloppes de frais courants
  • L'épargne
  • Les opérations spéciales
  • Les dépenses occasionnelles

Certaines séries ont été créées par l'assistant de création de séries, lors de l'initialisation de votre compte. Lors de cette phase de catégorisation, vous devrez certainement en créer de nouvelles, pour définir plus finement votre budget.

Une fois vos opérations catégorisées, vous pouvez vous rendre dans la vue Budget pour obtenir une vision d'ensemble de vos séries, dans chaque poste budgétaire.
