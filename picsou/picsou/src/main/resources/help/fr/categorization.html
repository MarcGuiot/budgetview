Pour calibrer votre budget, vous devez créer les enveloppes qui le caractérisent et y affecter les opérations que vous téléchargez du site internet de votre banque ou que vous saisissez vous-mêmes dans l'outil.

Dans la vue Catégorisation, BudgetView vous présente toutes les opérations que vous devez classer. Vous devez sélectionner ces opérations, puis les affecter à des enveloppes parmi cinq postes budgétaires distincts :

  • Les revenus
  • Les frais fixes
  • Les enveloppes de type variable (frais courants)
  • L'épargne
    • <
  • Les dépenses "extra" du mois

Certaines enveloppes ont été créées automatiquement la première fois que vous avez lancé l'application. Lors de cette phase de catégorisation, vous devrez certainement en créer de nouvelles, pour définir plus finement votre budget.

Une fois vos opérations catégorisées, vous pouvez vous rendre dans la vue Budget pour obtenir une vision d'ensemble de vos enveloppes, dans chaque poste budgétaire.
