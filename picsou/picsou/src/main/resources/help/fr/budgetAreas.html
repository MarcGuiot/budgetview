CashPilot organise votre budget en cinq postes budgétaires distincts :

Les revenus

Ce poste regroupe toutes vos sources de revenus, qu'elles soient régulières ou non.

Les types les plus courants de revenus sont les salaires, les revenus de professions libérales, les droits d'auteur, mais aussi tous les revenus d'investissements.

Les frais fixes

Ce poste regroupe les opérations qui se répètent régulièrement - tous les mois, tous les six mois, tous les ans, etc.

Les charges fixes les plus communes sont les loyers, les crédits, les abonnements, les impôts, les assurances. On peut également considérer comme charge fixe un achat réglé en plusieurs fois.

Les enveloppes de frais courants

Ce poste regroupe les frais courants, organisés en enveloppes générales telles que les courses, la santé, l'habillement, les loisirs.

Chaque mois, plusieurs opérations sont rattachées à ces enveloppes. Vous pouvez définir et suivre le montant total affecté chaque mois à chacune de vos enveloppes.

L'épargne

Ce poste regroupe les différents retraits ou dépôts effectués sur vos comptes d'épargne.

Les opérations "extra"

Ce poste regroupe les opérations exceptionnelles du mois - des achats importants, des voyages, des réparations exceptionnelles, etc.

Vous pouvez utiliser ce poste pour prévoir vos projets à l'avance - typiquement, vous pouvez créer une série "Achat TV" ou "Week-end à Etretat" dans l'un des mois à venir, pour anticiper les dépenses correspondantes et voir leur effet sur votre budget.
