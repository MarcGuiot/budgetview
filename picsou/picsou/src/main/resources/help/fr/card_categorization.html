Dans la vue Catégorisation, vous associez vos opérations bancaires à des séries réparties parmi plusieurs postes budgétaires.

Le bloc Opérations liste les opérations à classer.

Par défaut, la totalité des opérations est affichée dans cette liste. Lorsque le volume d'opérations devient trop important, vous pouvez limiter l'affichage en sélectionnant uniquement les opérations des mois sélectionnés, celles du dernier import, ou encore celles qui n'ont pas été classées.

Pour classer des opérations, il faut d'abord les sélectionner dans cette liste, puis ensuite choisir la série correspondante dans le bloc de droite.

Une fois l'opération sélectionnée, vous devez choisir le poste budgétaire, puis la série ou la catégorie à laquelle elle se rattache.

Dans chaque poste budgétaire, vous pouvez créer de nouvelles catégories en cliquant sur le bouton "+" situé au bas de l'écran. Dans le cas spécifique des Dépenses Occasionnelles, ce bouton "+" est remplacé par un bouton "liste" qui vous permet d'éditer la liste des catégories.
