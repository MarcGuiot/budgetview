La vue Budget présente l'état de toutes les enveloppes de vos comptes courants pour le mois sélectionné.

Chaque bloc présente les enveloppes actives pour le mois en cours.

Vous pouvez éditer chaque enveloppe en cliquant sur son nom, et ajouter de nouvelles enveloppes en utilisant le menu "Actions" au bas de chaque bloc.

Les flèches affichées sur la droite de chaque enveloppe vous indiquent l'évolution du montnant de l'enveloppe par rapport au mois précédent. Vous pouvez cliquer sur ces flèches pour obtenir des informations supplémentaires.

BudgetView vous montre le solde quotidien de vos comptes courants pour le mois sélectionné et le mois suivant.

Le solde minimum de ces comptes est affiché pour chaque mois, pour vous aider à régler finement les montants prévus de vos enveloppes.

Si ce solde est négatif dans les jours ou semaines à venir, vous devez réduire vos dépenses ou faire appel à votre épargne pour éviter le découvert. Au contraire, s'il est supérieur à 0, vous pouvez renforcer votre épargne ou bien vous autoriser quelques dépenses supplémentaires.

Consultez la page régler votre budget pour apprendre comment tirer le meilleur de cette vue.

Note: Le bloc "Epargne" affiche uniquement les enveloppes qui représentent des mouvements entre vos comptes courants et vos comptes d'épargne. Pour voir toutes les enveloppes d'épargne, vous pouvez cliquer sur le bouton "Voir tout" au bas de ce bloc.
