La vue Budget présente l'état de toutes vos séries :

Une fois vos opérations importées et catégorisées, vous utilisez cette vue pour régler vos séries en suivant les indications ci-dessous :

Comprenez votre balance et votre solde de fin de mois

Il s'agit de vos deux indicateurs principaux pour régler votre budget. Cliquez sur le bouton "loupe" pour accéder au détail de ces indicateurs :

Réglez les montants attendus pour vos séries

Cliquez sur les montants prévus pour les ajuster :

Réglez les périodicités des séries

Certaines séries ont des periodicités différentes des périodicités mensuelles classiques - par exemple un paiement de factures d'électricité qui survient tous les deux mois. Là encore, vous pouvez les éditer pour choisir la périodicité adaptée :

Créez les séries "extra"

Enfin, vous devez créer des séries dans le poste budgétaire "Extra" pour représenter les dépenses particulières qui vont marquer le budget de ce mois : achats importants, réparations, voyages, etc.

Vous utilisez aussi les séries "Extra" pour anticiper vos projets dans le futur :

Décalez des opérations

Il se peut que votre budget soit faussé par le fait qu'une opération importante ait été comptabilisée au mois précédent ou suivant celui où vous l'attendiez.

Le cas typique concerne les salaires, avec par exemple le salaire du mois de mai qui arrive au 1er juin, et celui du mois de juin qui arrive au 31 mai. Dans cette configuration, le budget de mai ne comporte aucun salaire tandis que celui de juin en comporte deux.

Pour corriger cette situation, vous devez vous rendre dans la vue Catégorisation, retrouver l'opération en question, et la décaler au bon mois en cliquant sur le bouton "Décaler".

Préparez les mois suivants

Une fois vos séries correctement réglées, vous pouvez repéter l'opération pour les mois suivants de manière à anticiper votre budget et préparer vos projets pour les mois à venir.
