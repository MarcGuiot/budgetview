Dans la vue Tableau de bord, BudgetView vous permet de créer et suivre vos projets.

Vous pouvez créer des projets pour tous types d'événements :

  • un voyage,
  • un mariage,
  • des travaux,
  • une naissance,
  • un achat important,
  • etc.

Au moment de créer un projet, vous détaillez l'ensemble des opérations qui lui seront associées. Par exemple, pour un voyage, vous précisez :

  • la réservation de l'hébergement,
  • l'achat de matériel (vêtements, équipement, photo, etc.),
  • le voyage aller,
  • l'hébergement,
  • l'alimentation,
  • les visites sur place,
  • les déplacements sur place,
  • le voyage retour,
  • etc.

BudgetView se charge alors de créer automatiquement une enveloppe correspondante, qui porte le même nom que le projet et se trouve dans le poste "Extras" de votre budget. Vous pouvez ainsi voir le montant attendu et réel pour ce projet dans le bloc "Extras" de la vue budget, et vous pouvez aussi affecter des transactions à cette enveloppe dans le bloc "Extras" de la vue Catégorisation.
