Pour utiliser Fourmics, vous devez tout d'abord le paramétrer correctement afin qu'il dispose d'une image aussi fidèle que possible de votre budget.

Pour cela, vous vous appuyez sur les relevés bancaires que vous pouvez télécharger sur le site Internet de votre banque. Sur la base des opérations trouvées dans ces relevés, vous définissez les séries d'opérations qui caractérisent votre budget :

  • vos revenus,
  • vos dépenses fixes (loyer, crédits, abonnements, etc.),
  • vos enveloppes de dépenses (courses, santé, habillement, loisirs, etc.),
  • vos opérations d'épargne,
  • le montant global que vous consacrez à des dépenses occasionnelles mineures.

Pour chacune de ces séries, vous indiquez quelle est sa périodicité (tous les mois, tous les deux mois, etc.), et vous pouvez également spécifier son échéancier complet pour les mois à venir.
