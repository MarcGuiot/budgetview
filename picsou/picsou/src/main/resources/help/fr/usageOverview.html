Pour utiliser BudgetView, vous devez tout d'abord le paramétrer correctement afin qu'il dispose d'une image aussi fidèle que possible de votre budget.

Pour cela, vous vous appuyez sur les relevés bancaires que vous pouvez télécharger sur le site Internet de votre banque. Sur la base des opérations trouvées dans ces relevés, vous définissez les enveloppes d'opérations qui caractérisent votre budget :

  • vos revenus,
  • vos dépenses fixes (loyer, crédits, abonnements, etc.),
  • vos dépenses variable (courses, santé, habillement, loisirs, etc.),
  • vos opérations d'épargne,
  • vos opérations extra (vacances, projets, imprévus).

Pour chacune de ces enveloppes, vous indiquez quelle est sa périodicité (tous les mois, tous les deux mois, etc.), et vous pouvez également spécifier son échéancier complet pour les mois à venir.

Une fois ces premières enveloppes convenablement configurées, BudgetView vous donne une vision générale de votre budget courant. A ce stade, vous pouvez déjà piloter votre budget mois par mois, en mettant à jour vos données chaque semaine.

Par la suite, vous pourrez utiliser BudgetView pour anticiper vos dépenses dans les mois à venir. Vous devrez pour cela d'ajuster la définition de votre budget pour obtenir une estimation réaliste de l'évolution de vos comptes. Une fois votre budget réglé, vous pourrez commencer à préparer vos projets - voyages, travaux, achats importants, etc.

Si vous n'avez pas encore importé vos données pour la première fois, nous vous recommandons de commencer par procéder au téléchargement.
