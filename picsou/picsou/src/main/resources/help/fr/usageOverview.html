Pour utiliser CashPilot, vous devez tout d'abord le paramétrer correctement afin qu'il dispose d'une image aussi fidèle que possible de votre budget.

Pour cela, vous vous appuyez sur les relevés bancaires que vous pouvez télécharger sur le site Internet de votre banque. Sur la base des opérations trouvées dans ces relevés, vous définissez les séries d'opérations qui caractérisent votre budget :

  • vos revenus,
  • vos dépenses fixes (loyer, crédits, abonnements, etc.),
  • vos enveloppes de dépenses (courses, santé, habillement, loisirs, etc.),
  • vos opérations d'épargne,
  • vos opérations extra (vacances, projets, imprévus).

Pour chacune de ces séries, vous indiquez quelle est sa périodicité (tous les mois, tous les deux mois, etc.), et vous pouvez également spécifier son échéancier complet pour les mois à venir.

Une fois ces séries convenablement configurées, CashPilot vous donne une vision générale de votre budget. A ce stade, cette vision est certainement grossière - il vous revient alors de l'affiner, en réglant manuellement les montants des échéanciers de certaines séries et en créant des séries "extra" pour les événements particuliers que vous prévoyez dans les mois à venir - projets, voyages, achats importants, etc.

Si vous n'avez pas encore importé vos données pour la première fois, nous vous recommandons de fermer cette fenêtre et de procéder au téléchargement.
