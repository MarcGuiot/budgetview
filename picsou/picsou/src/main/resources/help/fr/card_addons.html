La vue Extensions présente les extensions payantes de BudgetView.

Pour une description plus approfondie de ces extensions, vous pouvez consulter notre site internet :

http://www.mybudgetview.fr/extensions

Pour activer vos extensions une fois que vous les avez achetées, cliquez sur le bouton "Activer" à gauche de l'écran.
