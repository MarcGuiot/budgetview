Message de sécurité Windows 10

Lors de l'installation, Windows affiche un message d'avertissement indiquant que le logiciel n'est pas connu.

En effet, depuis Windows 10 Microsoft exige que les applications soient signées avec des certificats de sécurité payants proposés par certaines sociétés spécialisées.

Nous avons entamé les démarches pour obtenir ces certificats et publierons une nouvelle version du logiciel dès que ces démarches seront finalisées.

D'ici là, dans la fenêtre du fameux message "dangereux" vous trouverez un texte "plus de détails". En cliquant sur ce texte vous ferez apparaître un bouton "Installer quand même" qui vous permettra de continuer l'installation.

Application gratuite

La version de base est largement suffisante pour gérer votre budget. Des extensions payantes sont disponibles pour aller plus loin dans la maîtrise de vos projets.

Sûr et confidentiel

Vos données personnelles ne sont pas envoyées sur Internet. Vous les gardez sur votre ordinateur, et pouvez les crypter avec un mot de passe que vous êtes seul à connaître.

Pas de piège caché

Nous ne vous imposons aucune publicité. Nous ne vous espionnons pas. Nous ne vendons pas vos informations personnelles à des annonceurs.

Installation

Une fois le téléchargement terminé, double-cliquez sur le programme d’installation dans votre répertoire Téléchargements.

Mise en route

Pour utiliser BudgetView, des séances de 5 à 30 minutes, deux à cinq fois par mois, seront suffisantes pour mettre à jour et suivre votre budget.

Pour cette première utilisation, comptez toutefois entre 30 minutes et deux heures - vous allez devoir récupérer vos relevés d'opérations bancaires et faire un peu de tri pour définir correctement votre budget.
