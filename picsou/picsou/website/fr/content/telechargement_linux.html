Installez BudgetView
sur votre PC Linux.

Pour télécharger BudgetView, cliquez sur le bouton ci-dessous :

budgetview-2.21-fr.tar.gz

Prérequis

BudgetView nécessite la présence de Java sur votre système, version 1.5 et ultérieures. Pour utiliser BudgetView, vous devez vous assurer que la plateforme Java dont vous disposez est bien celle fournie par Oracle (la version OpenJDK n'est pas supportée).

Vous pouvez vérifier la version de Java de votre système en ouvrant un terminal et en lançant la commande suivante :

java -version

Vous obtiendrez un message du type :

java version "1.5.0_16"
Java(TM) 2 Runtime Environment, Standard Edition (build 1.5.0_16-b06-284)
Java HotSpot(TM) Client VM (build 1.5.0_16-133, mixed mode, sharing)

Si cette commande échoue ou que la version dont vous disposez est antérieure à 1.5, vous devez vous procurer la version la plus récente sur le site d'Oracle :

http://java.com/fr/download/manual.jsp

Si vous lisez l'anglais et utilisez l'une des distributions ci-dessous, vous pouvez également consulter les pages suivantes :

Ubuntu https://jdk-distros.dev.java.net/ubuntu.html
Debian https://jdk-distros.dev.java.net/debian.html
Gentoo http://www.gentoo.org/doc/en/java.xml

Guide d'installation

Enregistrez le fichier ci-dessus, en prenant soin de localiser le répertoire où vous l'avez placé.

Ouvrez un terminal, rendez-vous dans le répertoire en question et tapez la commande suivante pour décompresser le fichier :

tar xzvf budgetview-2.21-fr.tar.gz

Cela crée un répertoire BudgetView dans ce répertoire. Entrez la commande ci-dessous :

cd BudgetView

puis lancez BudgetView ainsi :

./BudgetView.sh

Voilà ! Vous êtes prêt à utiliser BudgetView.
