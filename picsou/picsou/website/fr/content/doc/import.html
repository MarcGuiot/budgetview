Importer vos relevés

Principe

Pour utiliser BudgetView, vous devez disposer de la liste des opérations bancaires qui ont été effectées sur vos comptes. La façon la plus rapide d'obtenir cette liste est de la télécharger sur le site internet de votre banque. Si vous ne disposez pas d'un tel accès, vous pouvez saisir vos opérations manuellement dans la vue Catégorisation.

Pour importer vos opérations via internet, vous suivez la démarche suivante :

1

Vous vous connectez au site internet de votre banque

2

Vous vous rendez sur la page de téléchargement

3

Vous sélectionnez les comptes et la période souhaités, ainsi que le format d'export (ces notions sont détaillées ci-dessous)

4

Vous lancez le téléchargement proprement dit

5

Si votre navigateur vous propose un choix entre "Ouvrir avec BudgetView" et "Enregistrer sous", vous pouvez choisir "Ouvrir avec BudgetView" : le fichier sera alors automatiquement redirigé vers BudgetView, où vous pourrez poursuivre l'import.

6

Si vous n'avez pas opté pour l'option "Ouvrir avec", vous devez enregistrer le fichier de relevés sur votre ordinateur, puis l'ouvrir à partir de BudgetView.

BudgetView fonctionne avec les principales banques Françaises, mais aussi avec la plupart des banques qui proposent un service de téléchargement en ligne.

Si votre banque ne figure pas dans la liste qui vous est proposée, sélectionnez la banque "Autre" - BudgetView fonctionnera normalement, seuls les libellés de vos opérations seront peut-être moins lisibles.

BudgetView reconnaît les deux formats de fichiers les plus couramment utilisés :

OFX

Format proposé pour des logiciels tels que Microsoft Money. Il est recommandé d'utiliser ce format, qui contient les informations de compte et de solde.

QIF

Format géré par le logiciel Quicken d'Intuit. Il ne doit être utilisé qu'en absence de fichiers OFX, car il ne contient pas d'informations sur le compte des opérations téléchargées, en particulier les soldes de ces comptes.

Dans un premier temps, vous pouvez vous contenter d'importer les opérations de vos comptes chèques. Vous pourrez récupérer vos autres comptes par la suite.

Pour établir un budget réaliste, il est recommandé d'importer environ un ou deux mois complets de données.

Banques supportées

BudgetView est aujourd'hui utilisé avec des relevés des banques suivantes :

  • BNP
  • Banque Populaire
  • Caisse d'Epargne
  • CIC
  • Crédit Agricole
  • Crédit Mutuel
  • La Poste
  • LCL
  • Société Générale

Cette liste n'est pas exhaustive. BudgetView peut parfaitement fonctionner avec les exports d'autres banques pourvu qu'elles proposent un export de vos relevés aux formats QIF et OFX.

Etape suivante

Une fois vos opérations importées, vous devez les catégoriser.
