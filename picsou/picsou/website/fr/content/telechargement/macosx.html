Guide d'installation

Si vous n'avez pas l'habitude d'installer des applications sur votre Mac, nous vous recommandons de suivre la démarche ci-dessous en vous assurant que vous visualisez cette page avec le navigateur Safari.

Pour commencer, cliquez sur le bouton "Téléchargement" ci-dessus. La fenêtre suivante devrait s'ouvrir :

Si elle ne s'ouvre pas, vous pouvez y accéder à partir du menu Fenêtres / Téléchargements.

Double-cliquez sur l'icône BudgetView. La fenêtre ci-dessous apparaît. Vous devez commencer par faire glisser l'icône de BudgetView sur votre dossier Applications :

Cela a pour effet de copier BudgetView sur votre machine. Pour accéder à la version que vous venez d'installer, double-cliquez sur votre dossier Applications :

Vous pouvez ensuite lancer BudgetView en cliquant sur son icône dans le dossier Applications :

Voilà ! Vous êtes prêt à utiliser BudgetView.
