Guide d'installation

Si vous n'avez pas l'habitude d'installer des applications sur votre ordinateur, ouvrez cette page avec le navigateur Firefox puis cliquez sur le lien ci-dessus. Vous verrez alors apparaître une fenêtre comme celle-ci :

Cliquez sur "Enregistrer". Vous devriez voir apparaître la fenêtre ci-dessous. Si la fenêtre n'apparaît pas, vous pouvez l'ouvrir à partir du menu "Outils / Téléchargements.

Double-cliquez sur BudgetView, une nouvelle fenêtre s'ouvre. Vous devez valider pour lancer l'installation. La question suivante vous sera alors posée :

Cliquez sur le bouton "OK".

Choisissez "oui". Le programme d'installation se lance :

Vous pouvez directement cliquer sur "Suivant", et suivre les consignes du programme d'installation.

Note : Si Java n'est pas installé sur votre machine, ce programme se chargera de procéder à son installation.

L'installation se termine comme suit :

Voilà ! Il ne vous reste qu'à cliquer sur "Terminer", et vous êtes prêt à utiliser BudgetView.
