Principaux changements

Au moment de lancer cette nouvelle version, vos données seront mises à jour comme suit :

  • Un compte sera associé à chacune de vos enveloppes sur la base des opérations qui leur ont déjà été affectées. Dans le cas où ces opérations concernaient plusieurs comptes, un groupe d'enveloppes sera automatiquement créé avec une enveloppe par compte.
  • Un compte sera affecté à chaque élément de projet sur la base des opérations déjà affectées. Dans le cas où des opérations de comptes différents auraient été assignées à un même élément de projet, celui-ci sera automatiquement dupliqué dans le projet de manière à ce que chaque élément ne concerne qu'un seul compte.

Afin que vous puissiez essayer cette version sans risque pour vos données, nous vous avons préparé une version spéciale de l'application présentée ci-dessous.

Pour éviter tout désagrément, nous vous encourageons à créer un fichier de sauvegarde de vos données avec votre application existante.

Application "Preview"

Nous mettons à votre disposition une version "visualisation seule" de l'application qui n'a pas besoin d'être installée et n'a aucune interaction avec votre répertoire de données. Celle-ci vous permettra de vérifier sans risque que la migration se passe bien.

Pour l'utiliser :

  1. Créez tout d'abord un fichier de sauvegarde de vos données avec votre application BudgetView existante. Notez bien l'endroit où vous placez cette sauvegarde sur votre ordinateur.
  2. Téléchargez la version "Visualisation seule" ici : Windows | Mac | Linux
  3. Lancez l'application en question en double-cliquant sur son icône dans le répertoire "Téléchargements" utilisé par votre navigateur.
  4. Restaurez votre fichier de sauvegarde pour retrouver vos données mises à jour.

Nous vous invitons alors à vérifier que vos enveloppes et projets sont corrects, et que les éventuels groupes d'enveloppés créés automatiquement vous conviennent.

Ajustement de votre budget

Les enveloppes créées par le logiciel lors de la mise à jour sont peut-être mal configurées - certaines informations ne peuvent en effet être déterminées de manière complètement automatique.

Vous serez ainsi peut-être amené à :

  • Ajuster les montants prévus de certaines enveloppes ou projets
  • Changer les noms d'enveloppes créées automatiquement
  • Modifier des éléments de projets

Si vous avez des questions ou rencontrez des difficultés, n'hésitez pas : contactez-nous !
