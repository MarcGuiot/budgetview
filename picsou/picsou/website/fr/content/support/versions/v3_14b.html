La prochaine version de BudgetView comporte deux évolutions importantes :

Suivi par comptes

Auparavant, tous vos comptes courants étaient fusionnés dans la vue Budget. Il vous était alors impossible d'anticiper le découvert d'un compte précis.

La nouvelle version de BudgetView remédie à cela en représentant individuellement chacun de vos comptes courants. Voici par exemple la nouvelle vue Budget :

Les graphiques de la vue Tableau de Bord sont eux aussi séparés par compte :

Cette évolution a une conséquence importante : chaque enveloppe ou chaque élément de projet doit dorénavant être associé à un et un seul compte courant.

Groupes d'enveloppes

Pour y voir plus clair dans votre budget, vous pouvez placer plusieurs enveloppes dans un même groupe comme suit :

Pour ajouter une enveloppe dans un nouveau groupe ou un groupe existant, cliquez sur son nom dans la vue Budget :

Mise à jour de vos données

Au moment de lancer cette nouvelle version, vos données seront mises à jour comme suit :

  • Un compte sera associé à chacune de vos enveloppes sur la base des opérations qui leur ont déjà été affectées. Dans le cas où ces opérations concernaient plusieurs comptes, un groupe d'enveloppes sera automatiquement créé avec une enveloppe par compte.
  • Dans la même logique, un compte sera affecté à chaque élément de projet sur la base des opérations déjà affectées. Dans le cas où des opérations de comptes différents auraient été assignées à un même élément de projet, celui-ci sera automatiquement dupliqué dans le projet de manière à ce que chaque élément ne concerne qu'un seul compte.

Pour que vous puissiez essayer cette version sans risque pour vos données, nous vous avons préparé une version spéciale de l'application présentée ci-dessous.

Application "Preview"

Nous mettons à votre disposition une version "visualisation seule" de l'application qui n'a pas besoin d'être installée et qui n'a aucune interaction avec votre répertoire de données.

Pour l'utiliser :

  1. Créez un fichier de sauvegarde de vos données avec votre application BudgetView existante. Notez bien l'endroit où vous placez cette sauvegarde sur votre ordinateur.
  2. Téléchargez la version "Visualisation seule" ici : Windows | Mac | Linux
  3. Lancez l'application en question en double-cliquant sur son icône dans le répertoire "Téléchargements" utilisé par votre navigateur.
  4. Restaurez votre fichier de sauvegarde pour retrouver vos données mises à jour.

Nous vous invitons alors à vérifier que vos enveloppes et projets sont corrects, et que les éventuels groupes d'enveloppés créés automatiquement vous conviennent.

Si tout fonctionne correctement, vous pouvez installer l'application en avant-première en téléchargeant le programme d'installation : Windows | Mac | Linux

La mise à jour automatique sera activée d'ici quelques semaines.
