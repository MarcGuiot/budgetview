Une toute nouvelle présentation

Au cours des derniers mois, nous avons repris la logique de l'ensemble de l'interface de manière à simplifier deux choses : la découverte de l'application par les nouveaux utilisateurs et la gestion individuelle des comptes. Cela a conduit à une toute nouvelle présentation :

Les principaux changements sont détaillés ci-dessous.

La météo de vos comptes

Gérer ses comptes, c'est tout d'abord répondre à la question : "Est-ce que l'un de mes comptes risque de passer dans le rouge ?"

BudgetView 4.0 répond directement à cette question :

Des prévisions vous sont proposées sur chacun de vos comptes : une météo couverte ou pluvieuse indique le risque que votre compte passe à découvert d'ici la fin du mois prochain.

Fitrage par comptes

Les vues Tableau de Bord, Budget et Comptes présentent dorénavant une même liste de comptes sur le bord gauche :

Vous pouvez afficher ou masquer le graphe de l'évolution du solde de chacun de vos comptes, mais aussi sélectionner un compte pour ne voir que les informations correspondantes dans le reste de la vue.

Avec ces évolutions, la vue spécifique des comptes d'épargne a disparu : les comptes d'épargne sont représentés exactement de la même manière que les comptes courants.

Multi-comptes plus souple

Depuis les origines de BudgetView, le calcul du prévisionnel était réalisé pour les comptes courants pris dans leur ensemble - il n'était pas possible de suivre l'évolution d'un compte courant en particulier.

Dans la version 3.14, nous avons permis de suivre chaque compte individuellement en associant chaque enveloppe à un compte donné. Il se trouve cependant que cette solution était trop contraignante pour certains de nos utilisateurs, obligeant dans certains cas à dupliquer plusieurs enveloppes.

La version 4.0 remédie à cela en proposant un mode beaucoup plus souple : les enveloppes sont par défaut associées à l'ensemble des comptes courants, et la projection dans le futur se base sur le comportement observé au cours des deux dernier mois.

Ainsi, si vous n'utilisez que l'un de vos comptes pour votre enveloppe Loisirs, le calcul du prévisionnel ne prendra en compte que ce compte-là. Si vous avez effectué un tiers des dépenses sur un compte et deux tiers sur un autre, cette même répartition s'appliquera dans la prévision.

Dans les cas où vous le jugez nécessaire, il vous est toujours possible d'éditer n'importe quelle enveloppe et préciser qu'elle ne s'applique qu'à un compte donné. Les projections se feront alors exclusivement sur ce compte, et l'enveloppe ne sera visible dans la vue Catégorisation que lorsque vous sélectionnerez une opération de ce compte.

Virements

Nous avons revu la façon de représenter vos virements entre vos comptes. Désormais, le bloc "Epargne" de la vue Budget devient "Virements" et accueille tous vos virements : virements vers ou à partir de vos comptes d'épargne, mais aussi virements entre vos comptes courants.

Une version de base gratuite

Pour faire profiter du meilleur budget prévisionnel à un plus grand nombre d'utilisateurs, nous rendons le coeur de BudgetView 4.0 gratuit. Certaines fonctionnalités deviennent des extensions payantes :

  • La constitution de groupes d'enveloppes dans la vue Budget
  • La gestion des projets
  • L'ensemble de la vue analyse
  • La visualisation du budget sur mobile Android

Ces extensions sont présentées dans une vue dédiée :

Si vous aviez acheté un code d'activation BudgetView pour les versions antérieures, celui-ci reste bien sûr valable et active automatiquement l'ensemble des extensions disponibles.

Installation

Deux points importants doivent être notés avant d'installer cette nouvelle version :

Cette version de BudgetView introduit des changements importants dans la façon dont votre budget est représenté au sein du logiciel. Nous avons intensivement testé l'ensemble, que ce soit à travers une batterie de plus de 1000 tests automatiques ou bien à travers nos propres tests manuels, mais par précaution nous vous recommandons de faire une sauvegarde de vos données avant d'installer la nouvelle version.

Par ailleurs :

Cette avant-première reprend l'ensemble des fonctionnalités de la version 3.14, à une exception près : les cartes à débit différé, pour lesquels il nous reste des éléments à finaliser.

Si vous utilisez une carte à débit différé, nous vous recommandons de ne pas installer cette version et d'attendre la version 4.0 finale.
