Peut-on utiliser plusieurs comptes ?

Il est possible de gérer plusieurs comptes à la fois dans BudgetView. Cela vaut pour deux types de comptes :

  1. Vous pouvez gérer plusieurs comptes bancaires pour le même budget. Pour cela, il vous suffit de les créer dans la vue Comptes ou d'importer les relevés correspondants.
  2. Vous pouvez créer plusieurs comptes utilisateur dans BudgetView, pour gérer des budgets différents. Pour cela, vous devez utiliser le menu "Fichier / Revenir à l'écran d'accueil, puis créer un nouveau compte utilisateur à partir de là.
