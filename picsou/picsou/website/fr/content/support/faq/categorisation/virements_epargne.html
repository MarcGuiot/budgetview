Comment classer les opérations
de virements d'épargne ?

Tout d'abord, vous devez déclarer vos comptes d'épargne dans BudgetView, soit en important les opérations correspondantes, soit en les créant vous-mêmes dans la vue Comptes.

Pour chaque compte d'épargne, BudgetView crée deux enveloppes :

  • une enveloppe pour les virements de vos comptes courants vers ce compte d'épargne
  • une enveloppe pour les virements de ce compte d'épargne vers vos comptes courants

Par exemple, si votre compte d'épargne se nomme "Livret A", vous trouverez deux enveloppes dans le bloc "Epargne" de la vue Budget :

  • "Vers Livret A"
  • "De Livret A"

Pour classer vos opérations de virement, vous devez sélectionner l'enveloppe correspondante dans le bloc "Epargne" de la vue Catégorisation.
