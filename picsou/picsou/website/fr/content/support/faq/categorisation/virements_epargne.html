Tout d'abord, vous devez déclarer vos comptes comme étant des comptes d'épargne, soit au moment où vous importez vos opérations, soit en cliquant sur ces comptes dans la vue "Comptes".

Création des enveloppes

Pour chaque compte d'épargne, BudgetView crée automatiquement deux enveloppes :

  • une enveloppe "Vers le compte [...]" pour les virements de vos comptes courants vers ce compte d'épargne
  • une enveloppe "A partir du compte [...]" pour les virements de ce compte d'épargne vers vos comptes courants

Par exemple, si votre compte d'épargne se nomme "Livret A", vous trouverez deux enveloppes dans le bloc "Epargne" de la vue Budget :

  • "Vers Livret A"
  • "De Livret A"

Si vous le souhaitez, vous pouvez créer vous mêmes d'autres enveloppes pour prévoir des mouvements concernant vos comptes d'épargne.

Veuillez noter que vous ne voyez dans la vue Budget que des enveloppes qui concernent votre compte courant, car toute cette vue est centrée sur vos comptes courants. Si vous voulez gérer des enveloppes qui ne concernent que vos comptes d'épargne, vous trouverez dans le bloc "Epargne" de la vue Budget un bouton "Voir tout" qui vous permettra d'accéder à l'ensemble de vos enveloppes d'épargne.

Catégorisation des opérations d'épargne

Pour classer vos opérations de virement, vous devez vous rendre dans la vue Catégorisation, puis cliquer sur les opérations en question et choisir le poste budgétaire "Epargne". Vous devriez voir alors s'afficher à droite les enveloppes correspondantes.
