Les opérations de virement sont gérées dans BudgetView à travers des enveloppes spécifiques placées dans le poste budgétaire "Virements".

Chaque enveloppe de virement représente un mouvement d'un compte vers un autre, dans un seul sens. Cela a deux conséquences :

  1. Pour toutes les enveloppes de ce poste, vous devez indiquer le compte source et le compte destination.
  2. Pour gérer des échanges dans les deux sens, vous devez créer deux enveloppes, une dans chaque sens.

Pour catégoriser une opération de virement, vous devez vous rendre dans le poste Virements de la vue Catégorisaton. Seules les enveloppes qui ont pour compte source le compte de cette opération seront affichées.
