Comment utiliser l'application Android ?

L'application BudgetView pour Android fonctionne en complément de l'application "bureau", en vous permettant de visualiser sur votre terminal mobile le budget que vous avez préparé sur votre ordinateur.

Pour l'utiliser, vous devez vous rendre dans le menu "Fichier / Accès mobile Android". Au cours de cette phase d'activation, deux emails vous seront envoyés :

  • un message de confirmation de l'activation de votre compte mobile,
  • un message contenant le lien de téléchargement de l'application Android. Celle-ci n'est en effet pas encore disponible sur Google Play, vous devrez l'installer manuellement sur votre smartphone.
