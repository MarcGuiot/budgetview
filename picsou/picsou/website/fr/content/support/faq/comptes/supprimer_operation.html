Comment supprimer une opération ?

Dans la vue "Catégorisation" ou la vue "Comptes", cliquez sur l'opération avec le bouton droit de votre souris. Vous verrez apparaître un menu avec l'option "Supprimer...".

Vous pouvez aussi appuyer sur la touche "Supprimer" de votre clavier une fois l'opération sélectionnée.
