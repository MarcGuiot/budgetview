Vous pouvez transmettre les informations de votre budget d'un ordinateur à l'autre, même si les deux ordinateurs utilisent des systèmes d'exploitation différents (Mac/Windows/Linux).

Pour cela, vous devez avoir acquis une clef d'activation du logiciel. Il vous suffit alors de suivre les étapes ci-dessous.

Sur votre premier ordinateur :

  • Lancez BudgetView et utilisez la fonction "Créer un fichier de sauvegarde" du menu Fichier
  • Copiez ce fichier de sauvegarde sur une clef USB, ou bien envoyez-vous le fichier par email

Sur votre nouvel ordinateur :

  • Récupérez le fichier de sauvegarde
  • Installez BudgetView
  • Lancez BudgetView
  • Enregistrez votre code d'activation (en savoir plus)
  • Utilisez la fonction "Restaurer une sauvegarde" du menu Fichier

NB : La clef d'activation de BudgetView vous donne le droit d'utiliser l'application sur deux ordinateurs différents. Prenez garde toutefois à ne pas donner vos identifiants à un tiers, car votre clef pourrait être désactivée.
