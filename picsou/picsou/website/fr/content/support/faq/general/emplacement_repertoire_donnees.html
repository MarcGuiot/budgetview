Où sont stockées les données et les traces de BudgetView sur ma machine ?

Vous pouvez accéder au répertoire de stockage de BudgetView soit pour effectuer des sauvegardes des données, soit pour accéder aux traces de l'application.

Ce répertoire est stocké à l'emplacement suivant, selon votre système d'exploitation :
Mac /Users/{votre nom d'utilisateur}/Library/Application\ Support/BudgetView/
Windows C:\Users\{votre nom d'utilisateur}\Application Data\BudgetView
ou
C:\Users\{votre nom d'utilisateur}\AppData\Local\BudgetView
Linux $HOME/.budgetview

En cas de problème technique, vous trouverez dans ce répertoire un répertoire "logs" dans lequel se trouve un fichier log.txt. N'hésitez pas à joindre ce fichier lorsque vous contactez notre support, nous pourrons y voir les raisons d'éventuels problèmes.

Si vous le souhaitez, vous pouvez déplacer ce répertoire de stockage de données BudgetView, comme indiqué dans la page déplacer le répertoire de données.
