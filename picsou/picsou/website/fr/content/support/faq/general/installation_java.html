Au moment d'installer BudgetView, il vous sera certainement demandé d'installer Java.

Nous vous recommandons d'accepter cette installation qui est indispensable pour que BudgetView puisse fonctionner sur votre ordinateur. Mis à part l'espace disque utilisé (de l'ordre de 150Mo), elle ne devrait avoir aucune conséquence négative sur votre système.

Quelques informations sur cette technologie

Java est une plateforme technique qui permet à des applications de fonctionner sur la plupart des systèmes d'exploitation (Windows, Linux, Mac) sans nécessiter de développements spécifiques à chaque système. C'est l'une des technologies les plus largement répandues au monde, notamment pour le développement d'applications en entreprise. Elle est par exemple au coeur de Twitter. Elle est aujourd'hui portée par Oracle et IBM.

Java est cependant moins connu comme support d'applications grand public, avec toutefois des exceptions telles que Minecraft ou Runescape.

Depuis quelques années, Java a mauvaise presse en termes de sécurité, mais cela ne concerne qu'une partie très spécifique de cette plateforme : le "java plugin" utilisé par les navigateurs pour faire fonctionner ce que l'on appelle des "applets". Cette technologie obsolète n'est toutefois plus activée par défaut dans les navigateurs récents, ce qui vous met à l'abri des attaques de ce type.

Comment l'installer ?

Sous Windows, Java sera installé automatiquement par notre logiciel d'installation. Vous pouvez également l'installer vous-même à partir du site d'Oracle.

Sous Mac, les dernières versions de Mac OS X vous proposent automatiquement d'installer Java lorsque vous lancez BudgetView. En cas de problème, vous pouvez sinon l'installer à partir du site d'Apple.

Sous Linux, vous pouvez l'installer vous-même à partir du site d'Oracle.
