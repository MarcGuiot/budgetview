Après avoir installé sur votre Mac la dernière version de Mac OS X appelée Yosemite, vous devriez voir apparaître au lancement de BudgetView le message suivant :

Yosemite installe effectivement une nouvelle version de Java avec laquelle BudgetView n'est pas compatible. Nous allons devoir modifier le système d'installation de BudgetView sur Mac pour tenir compte de cette spécificité, mais cela prendra un peu de temps.

Pour pouvoir utiliser BudgetView à nouveau, nous vous recommandons d'installer Java 6 à l'aide du programme d'installation spécifique fourni par Apple. Il vous suffit pour cela de cliquer sur le bouton "Télécharger" en haut de la page ci-dessous, puis de suivre les consignes à l'écran :

http://support.apple.com/kb/DL1572?viewlocale=fr_FR&locale=fr_FR
