Dans BudgetView, les opérations récurrentes se gèrent à travers la mécanique des enveloppes. Chaque enveloppe représente justement un poste de recette ou de dépense récurrent, dont vous pouvez régler la périodicité et le montant dans les mois à venir.

Visualiser les opérations planifiées

BudgetView génère automatiquement des opérations planifiées dans les mois à venir pour toutes les enveloppes déclarées dans votre budget. Vous pouvez visualiser ces opérations en sélectionnant l'option "Afficher les opérations prévues" dans la vue Comptes :

Régler la récurrence et les dates de début/fin

Par défaut, les enveloppes se répètent avec le même montant chaque mois, à l'infini. Pour modifier une enveloppe, vous pouvez cliquer sur son nom dans la vue Budget :

La fenêtre d'édition qui apparaît vous permet alors de modifier la périodicité, ainsi que de fixer une date de début ou de fin pour l'enveloppe :

Régler le jour de prélèvement

Par défaut, BudgetView génère dans le futur un profil de revenus ou de dépenses similaire à ce qui a été observé dans les derniers mois :

  • pour les enveloppes à opération unique telles que celles du poste Fixe (crédits, etc.) : une seule opération est générée par défaut le 15 du mois.
  • pour les enveloppes regroupant plusieurs opérations, notamment celles du poste Variable (courses, loisirs, etc.) : plusieurs opérations réparties dans le mois.

Pour affiner les prévisions, vous pouvez forcer BudgetView à génere une seule opération mensuelle pour une enveloppe donnée, par exemple pour votre salaire ou vos crédits. Il vous faut pour cela vous rendre dans l'onglet "Planification" de la fenêtre d'édition d'enveloppe :
