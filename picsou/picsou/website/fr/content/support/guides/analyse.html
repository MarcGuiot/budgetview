BudgetView vous aide à explorer votre budget pour mieux comprendre comment il se compose et comment il évolue, mais aussi pour retrouver des éléments précis. Cette page présente quelques principes de base qui vous permettront de trouver votre chemin.

Voir plusieurs mois

Vous pouvez sélectionner plusieurs mois simultanément en cliquant sur l'un d'eux, puis en déplaçant votre curseur de souris tout en maintenant votre bouton enfoncé :

Quand plusieurs mois sont ainsi sélectionnés :

  • Les vues Comptes et Catégorisation affichent les opérations des mois sélectionnés.
  • la vue Budget affiche la somme des montants pour les mois en question.

Vous pouvez également cliquer sur le libellé d'une année pour sélectionner tous les mois correspondants :

Rechercher des opérations

Le champ de recherche situé en haut à droite dans la vue Comptes vous permet de rechercher des opérations par leur nom, leur enveloppe ou leur montant :

Combinez cette recherche à la multi-sélection de mois pour chercher des opérations sur une plage de temps donnée.

La vue analyse

Cette vue vous permet d'explorer la constitution et l'évolution des enveloppes qui composent votre budget.

Vous pouvez naviguer dans les enveloppes de votre budget en cliquand sur les graphes de cette vue :

Les liens en haut de ce block permettent de remonter dans la hiérarchie des envloppes.

Cliquez sur le bouton "Afficher le tableau des détails" au bas de l'écran pour faire apparaître le tableau des montants de chaque enveloppe. Dans ce tableau, vous pouvez cliquer sur le montant d'une enveloppe pour éditer les caractéristiques de l'enveloppe - typiquement pour modifier le montant attendu, ou pour changer la périodicité de l'enveloppe.
