A l'issue du premier import de vos relevés et de la première catégorisation de vos opérations, vous obtenez une définition de budget qui reste somme toute assez grossière.

A ce stade, vous devez continuer à affiner la vision de votre budget, à travers les activités décrites ci-dessous.

1. Réglez les montants attendus de vos enveloppes variables

Chaque enveloppe définit un échéancier de paiements, qui permet d'anticiper vos dépenses pour le mois en cours et les mois à venir.

Pour les enveloppes fixes nouvellement créées, le calcul de cet échéancier est automatique : le montant prévu pour un mois donné est basé sur celui prévu pour le mois précédent.

En revanche, pour les enveloppes variables, il vous revient de régler le montant que vous souhaitez allouer, enveloppe par enveloppe. Pour changer un montant attendu, il vous suffit de cliquer dessus dans la vue Budget.

2. Définissez les périodicités de vos enveloppes

Certaines enveloppes ont des periodicités différentes des périodicités mensuelles classiques. Là encore, vous pouvez les éditer pour choisir une périodicité adaptée :

  • tous les deux mois
  • tous les trois mois
  • tous les six mois
  • tous les ans

Voici une liste des enveloppes les plus fréquemment ajustées :

  • Les anniversaires de vos proches, qui se répètent... tous les ans.
  • Les révisions de vos véhicules, qui se répetent environ tous les ans.
  • Les factures de gaz et d'électricité, qui se répetent habituellement tous les deux mois.
  • Les impôts, dans le cas où vos règlements ne sont pas mensualisés
  • Certaines assurances, par exemple les assurances habitation, qui se répètent tous les ans
  • Les cadeaux de Noël
  • Les revenus irréguliers (13e mois, primes, etc.)

3. Ajoutez les enveloppes "extra" et les projets manquants

En début de mois, vous créez les enveloppes correspondant aux dépenses "extra" que vous avez déjà prévues : anniversaires, sorties, dépenses loisirs importants, mobilier, etc.

Pour les dépenses plus complexes - voyages, travaux, mariages - vous pouvez créer des "projets" dans lesquels vous anticipez les différentes dépenses élémentaires. Ce point est détaillé plus loin dans ce guide.

4. Ajustez votre épargne

Si vous avez décidé de renforcer votre épargne, que ce soit pour amortir un imprévu, ou pour préparer un projet, vous pouvez l'indiquer dans BudgetView pour compléter votre budget.

Dans cette optique, nous vous recommandons de suivre l'une des grandes règles de la gestion de finances personnelles : commencer par épargner, avant de procéder aux dépenses courantes du mois. Cette méthode est bien plus efficace que celle qui consiste à espérer qu'en fin de mois il reste de l'argent à épargner sur vos comptes.

Pour cela, une solution éprouvée consiste à mettre en place un virement automatique de vos comptes courants vers un compte d'épargne. Ce virement doit intervenir en début de mois, juste après que vous ayez perçu vos revenus.

Une fois le virement mis en place avec votre banque, vous pouvez le déclarer dans BudgetView en vous assurant que votre compte d'épargne est bien déclaré, et en réglant dans le bloc "Epargne" le montant prévu de l'enveloppe qui représente les virements de vos comptes courants vers ce compte.

5. Equilibrez votre budget

Pour régler votre budget, BudgetView vous montre l'évolution quotidienne du solde quotidien de vos comptes courants, pour le mois courant et les mois à venir :

Le chiffre clef de cet indicateur est le solde minimum dans les semaines à venir :

  • Si ce solde minimum est négatif, vous devrez payer des frais de découvert auprès de votre banque. Vous devez alors réduire certaines enveloppes, typiquement des enveloppes variables telles que les courses ou les loisirs, ou bien prélever de l'argent de votre épargne.
  • A l'inverse, si ce solde est largement positif, vous pouvez vous autoriser des dépenses supplémentaires, ou renforcer votre épargne.

6. Pilotez votre budget

Une fois votre budget correctement réglé, vous le pilotez chaque semaine en procédant comme suit :

  • Vous importez vos dernières opérations
  • Vous classez les quelques opérations qui n'ont pas été classées automatiquement
  • Vous voyez immédiatement dans la vue budget les éventuels dépassements par rapport à ce que vous aviez prévu. Vous pouvez alors réagir en conséquence et ajuster vos dépenses pour éviter les découverts et protéger votre épargne.

7. Préparez votre budget prévisionnel

Les enveloppes que vous utilisez mois après mois vous aident également à préparer votre budget prévisionnel. Il vous suffit de sélectionner les mois à venir et de modifier les montants prévus des enveloppes pour affiner les projections. Vous pouvez alors préparer le futur, et en particulier vos projets.

Lorsque vous êtes à l'aise dans le pilotage de votre budget, vous pouvez commencer à préparer vos projets.
