Pour repérerer d'éventuelles opérations erronées ou frauduleuses, vous pouvez soit parcourir rapidement vos relevés, soit pointer les opérations une à une.

Parcourir les dernières opérations importées

Dans la vue Catégorisation, sélectionnez l'option d'affichage "Dernier fichier importé". Vous pouvez alors parcourir les opérations et vous assurer qu'elles sont toutes correctes.

Pointer les opérations

Vous pouvez pointer les opérations une à une pour comparer par exemple ce qui est dans BudgetView avec vos relevés bancaires ou vos factures. Pour faciliter cela, vous pouvez sélectionner l'option "Afficher le pointage des opérations" dans le menu Affichage. Vous verrez apparaître une nouvelle colonne dans le tableau qui se trouve à droite de la vue Catégorisation :

Vous pouvez alors cliquer sur les pastilles devant chaque opération au fur et à mesure que vous les pointez.

Notez que certaines opérations seront certainement déjà pointées la première fois que vous ferez apparaître la colonne de pointage. Cela est dû au fait que les opérations sont automatiquement pointées lorsque vous les catégorisez manuellement.
