Derniers changements

version 2.20 29 décembre 2011

  • Correction d'une erreur dans certains calculs des soldes des comptes
  • Correction d'un problème lors d'un double "Cmd-Q" sous Mac OS X
  • Les transactions créées manuellement ne sont plus pointées automatiquement lors de la catégorisation

Pour profiter de ces évolutions, il vous suffit de relancer BudgetView, attendre quelques secondes, puis suivre les instructions qui s'affichent. Si vous n'avez pas encore BudgetView, installez-le dès maintenant !

version 2.17 29 novembre 2011

  • Correction d'un blocage lorsque l'on quitte l'application à partir de la page d'accueil sous Mac OS X
  • Correction d'un problème dans la navigation d'une enveloppe de la vue Budget vers les opérations correspondantes dans la vue Comptes
  • Elimination des "tooltips" fantômes lors de l'utilisation de Alt-Tab sous Mac OS X
  • Ajout d'une dizaine de banques Françaises, notamment des banques en ligne

version 2.14 31 octobre 2011

  • Import des catégories dans les fichiers Money
  • Correction d'une erreur survenant lors de l'import d'opérations pour un compte ayant été supprimé.
  • Correction d'une erreur survenant dans certains cas de suppression d'enveloppe (merci Bernard)

version 2.13 18 octobre 2011

  • Clarification de l'écran d'import dans le cas où il y a plusieurs comptes dans un même fichier.
  • Correction d'un défaut qui pouvait causer l'invalidation répétée du code d'activation.
