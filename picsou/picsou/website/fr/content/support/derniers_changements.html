Derniers changements

version 2.27 16 avril 2012

  • Si vous souhaitez mieux contrôler les prévisions faites par BudgetView pour vos dépenses futures, vous pouvez désormais forcer une enveloppe donnée à ne prévoir qu'une opération par mois, à jour fixe. Cela est utile par exemple pour indiquer que votre salaire est prévu le 29 de chaque mois. Pour régler tout cela, cliquez sur le nom de l'enveloppe en question dans la vue Budget et rendez-vous dans l'onglet "Planification".
  • Vous pouvez également modifier le libellé des opérations, en cliquant avec le bouton droit dans les tables des vues Catégorisation et Comptes.
  • Lorsque vous importez vos fichiers, ou lorsque vous créez un compte, vous pouvez maintenant ajouter une banque à la liste.
  • Au moment de saisir une opération, il vous est dorénavant possible de choisir un mois dans le passé, même si ce mois n'est pas affiché dans la barre de sélection de mois en haut de l'écran.
  • Cette version corrige également quelques dysfonctionnements, notamment dans l'import de fichiers CSV et dans l'affichage des projets.

Pour profiter de ces évolutions, il vous suffit de relancer BudgetView, attendre quelques secondes, puis suivre les instructions qui s'affichent. Si vous n'avez pas encore BudgetView, installez-le dès maintenant !

version 2.25 13 mars 2012

  • Vous pouvez dorénavant importer vos fichiers CSV, par exemple produits à partir d'un tableur comme Excel ou Numbers. Pour plus d'informations, consultez le guide d'import CSV.
  • Deux nouvelles options sont disponibles dans le menu des préférences :
    • Modifier le format utilisé pour l'affichage des dates
    • Modifier le thème de couleurs de l'application
  • Lors de la première utilisation de l'application, il vous est proposé de voir BudgetView à l'oeuvre à l'aide d'un compte de démonstration.
  • Cette version corrige également quelques dysfonctionnements, notamment dans la suppression manuelle d'opérations.
