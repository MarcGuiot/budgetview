After installing and launching BudgetView on Mac OS X 10.10 (Yosemite) or 10.11 (El Capitan), you will be asked for permission ton install Java 6 on your computer. You must accept this installation for BudgetView to work.

However, this default installation is not fully compatible with BudgetView. To run BudgetView successfully, you must install an updated version provided by Apple at the following address:

http://support.apple.com/kb/DL1572?viewlocale=fr_FR&locale=fr_FR
