Installation guide

If you're not used to installing applications on your computer, open this webpage with the Firefox browser and click on the "Download" button above. You will see a windows like this:

Click on "Save File" - you should then see a window like the one below. If the window does not appear, you can open it from the "Tools / Downloads" menu.

Double-click now on BudgetView. A new window appears. You have to accept to start the installation process, then Windows will as you to confirm the execution of the installation program. Once you accept the installation, the following dialog will appear:

Select "Yes". The installation begins:

At this stage you can select "Next" and follow the instructions of the installation program.

Note: If Java is not installed on your computer, the BudgetView installer will guide you through the installation.

The BudgetView installation completes like this:

Voilà! You just have to click on "Finish", and you're ready to use BudgetView.
