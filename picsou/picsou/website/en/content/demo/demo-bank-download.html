Welcome to our Demo Bank!

You will find here instructions to download a sample OFX file to try out BudgetView:

If BudgetView is properly installed, you can then import this sample file as follows, depending on which browser you use:

Firefox

  1. Click on this link: demo.ofx
  2. Select the "Save File" option
  3. In the "Downloads" window, double click on the "demo.ofx" file (you can open this window from the "Tools/Downloads" menu if needed).
  4. This should start BudgetView automatically. If BudgetView does not appear, select it in your task bar.

Safari

  1. Click on this link: demo.ofx
  2. In the "Downloads" window, double click on the "demo.ofx" file (you can open this window from the "Window/Downloads" menu if needed).
  3. This should start BudgetView automatically.

Chrome

  1. Click on this link: demo.ofx
  2. The downloaded file should appear at the bottom of the window.
  3. Click on the file at the bottom of the window, that will open BudgetView automatically.

Troubleshooting

If double clicking on the file does not open BudgetView, you can right-click on the file in your browser "Downloads" window and select "Show in finder" to locate the file. You can then right click on the file, use "Open with..." and find BudgetView.
